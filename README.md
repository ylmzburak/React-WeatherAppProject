🌤️ React Weather App

A simple weather application built with **React + Vite** that allows users to search for a city and view current weather information.

🚀 Live Demo

https://react-weather-app-project-t6fr.vercel.app

📌 Features

* Search weather by city
* Displays temperature and weather conditions
* Responsive UI
* Data fetched from WeatherAPI

🛠️ Technologies

* React
* Vite
* JavaScript
* CSS
* WeatherAPI
* Vercel
